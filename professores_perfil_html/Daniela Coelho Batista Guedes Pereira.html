2007 - 2008
ELABORAÇÃO DE UM SISTEMA INFORMATIZADO PARA ESTUDOS FENOLÓGICOS NA RESERVA BIOLÓGICA GUARIBAS, PARAÍBA, BRASIL
Descrição: O presente projeto tem como objetivo construir um sistema informatizado para o acompanhamento dos dados fenológicos e de polinização levantados pelo projeto ?Estudos Fenológicos, Florísticos, de Polinização e Dispersão na Reserva Biológica Guaribas, Paraíba, Brasil?. Neste projeto pretende-se produzir um sistema de suporte à decisão para o gerenciamento de dados fenológicos e de polinização, consolidando estes dados e subsidiando estudos futuros. E assim possibilitar a melhor compreensão e conseqüente aplicação de tais enfoques à ecologia pelos botânicos, o que provavelmente levará a uma ampliação no estudo do ecossistema da Mata Atlântica..
