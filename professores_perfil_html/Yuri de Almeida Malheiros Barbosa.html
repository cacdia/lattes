2019 - 2021
Aumento de Dados Textuais para Aprendizagem de Máquina
Descrição: Para conseguir bons resultados, técnicas de aprendizagem de máquina geralmente precisam de uma grande quantidade de dados. Entretanto, a obtenção do volume de dados necessário pode ser muito custosa ou inviável. Com isso, técnicas para treinamento de modelos de aprendizagem de máquina que exijam uma quantidade menor de dados têm sido investigadas. Em visão computacional, o aumento de dados, que consiste em gerar dados artificiais baseados num conjunto pequeno de dados existentes, é uma das técnicas mais difundidas para expandir os dados de treinamento. Por outro lado, em processamento de linguagem natural, o aumento de dados ainda é pouco explorado. Assim, este projeto de pesquisa tem como objetivo pesquisar, analisar e propor técnicas de aumento de dados textuais, para descobrir como e quanto essas técnicas podem melhorar tarefas de processamento de linguagem natural..
Situação: Concluído; Natureza: Pesquisa.
Alunos envolvidos: Graduação: (2) .
