FlexTV - Desenvolvimento de Middleware para o Sistema Brasileiro de Televisão Digital
Descrição: Em um sistema de televisão digital, o subsistema de difusão e acesso é constintuído pelos módulos que propiciam a geração e empacotamento das informações a serem transmitidas para os usuários, enquanto os módulos contidos no subsistema terminal de acesso efetuam o processamento necessário para os usuários possam desfrutar das informações transmitidas. Um dos componentes do terminal de acesso do sistema de televisão digital é o middleware. As principais funções de um middleware para tlevisão digitial interativa são: possibilitar a execução de aplicações, fornecendo para as mesmas um conjunto de APIs bem definido, abstraindo características específicas de hardware e de sistema operacional; e fornecer serviços para essas aplicações tais como, serviços de comunicação, acesso a fluxos elementares de áudio, vídeo e dados. O projeto FlexTV é um dos subprojeto do projeto do Sistema Brasileiro de Televisão Digital. O FlexTV tem por objetivo principal a concepção de uma arquitetura conceitual de um middleware e a definição de APIs para programas de televisão interativos que serão avaliadas pelo Governo Brasileiro e, caso aprovadas, utilizadas como parte do Modelo de Referência do Sistema Brasileiro de Televisão Digital. O projeto prevê a elaboração de análises tecnológicas e econômicas de modo a subsidiar a definição de um modelo de referência adequado às necessidades da população brasileira. A arquitetura e as APIs propostas serão validadas através da implementação de um protótipo do middleware e de aplicações residentes no terminal de acesso do sistema de televisão digital. O projeto prevê ainda a realização de testes de integração do middleware com resultados de outros sub-projetos do projeto do Sistema Brasileiro de Televisão Digitral: aplicações, serviço de saúde, transmissão, camada de transporte, terminal de acesso, entre outros..

I2TV - Infraestrutura Internet2 para Desenvolvimento e Teste de Ferramentas e Programas para Televisão Interativa
Descrição: O projeto I2TV tem por objetivo dar suporte a formação de um consórcio para desenvolvimento e teste de aplicações e ferramentas para TV Interativa em ambiente Internet2. Uma série de questões permanecem em aberto em relação ao tema TV Interativa. A principal delas é a própria definição do que é um serviço de TV Interativa. No contexto deste projeto televisão interativa é definida como a junção das tecnologias associadas à difusão de vídeo com tecnologias interativas viabilizadas por um "canal de retorno" ligando terminais de acesso avançados com as emissoras de televisão. A maioria dos padrões e produtos que estão sendo propostos e implementados apontam claramente para uma integração de tecnologias de broadcast de vídeo digital com a WWW. O conjunto de aplicações que surgirão, ou serão estendidas, como conseqüência desta integração é bastante vasto. Alguns exemplos são comércio eletrônico via TV, ou t-comerce, broadcasting de vídeo na Internet, vídeo sob demanda etc. Exemplos de aplicações que serão estendidas incluem programas de auditório com participação em tempo real dos telespectadores de suas casas, TVs comunitárias interativas, versões interativas de programas de televisão educacionais como o Telecurso etc..
