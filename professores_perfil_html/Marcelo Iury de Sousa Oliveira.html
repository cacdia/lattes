Sistema de Apoio à Decisão para Análise da Evasão do Ensino Superior
Descrição: Dados abertos são dados disponibilizados na web passíveis de utilização, reutilização e compartilhamento por qualquer pessoa. Com a criação da Lei de Acesso à Informação em 2011, cada vez mais dados governamentais têm sido publicados, mas de forma bruta e de difícil entendimento. Para auxiliar na análise de grande volume de dados surgem as ferramentas de Business Intelligence, dentre elas estão os Sistemas de Apoio à Decisão, que são ferramentas completas, de fácil uso e que torna a análise mais rápida e eficaz. Sendo o processo de tomada de decisão de extrema importância, tanto em organizações privadas como em públicas, existe a necessidade de criação de ferramentas para o apoio a tomada de decisão no Ensino Superior. Nesse contexto, um fenômeno social que vem sendo amplamente discutido é a evasão, o qual tem alcançado índices de aproximadamente 50%. Neste sentido, este estudo tem como objetivo criação de um Sistema de Apoio à Decisão para o auxílio no processo de tomada de decisão sobre as taxas de Evasão no ensino superior, utilizando dados do Censo da Educação Superior nos últimos dez anos. Além disso, busca-se também verificar sua correlação com outras fontes de dados, como por exemplo, os dados do Exame Nacional de Desempenho dos Estudantes. Para isto, o método utilizado no desenvolvimento seguirá a abordagem bottom up, sendo compreendida em oito etapas, as quais são: planejamento, levantamento de necessidades, modelagem dimensional, projeto físico do banco de dados, projeto de extração, transformação e carga, desenvolvimento de aplicações, validação e teste e, por fim, implantação da solução. Espera-se que ao final deste projeto seja possível fornecer dados atualizados para a tomada de decisão no que diz respeito à evasão no ensino superior..

Melhores Práticas e Serviços para o Desenvolvimento do Ecossistema de Dados Abertos da Cidade de Recife
Descrição: Nos últimos anos, governos de diversos países têm demonstrado grande interesse em disponibilizar seus dados em formato aberto com o intuito de oferecer maior transparência aos cidadãos. A ideia principal desta iniciativa consiste em oferecer dados de forma livre e pública na Web, a fim de que os cidadãos possam ter fácil acesso aos dados produzidos pelo governo. Abrir dados públicos fortalece instituições, garante cidadania e controle social, combate a corrupção, promove a transparência, a fiscalização e a geração de ideias criativas em políticas públicas pela própria sociedade. Dados Abertos Governamentais, ou seja, dados produzidos pelo governo e disponibilizados em formato aberto, podem ser facilmente encontrados nos chamados portais de Dados Abertos, os quais oferecem uma interface para catalogação e acesso aos dados. A fim de alcançar o seu pleno potencial econômico, uma iniciativa de Dados Abertos deve fazer parte de um ecossistema composto, principalmente, por provedores e consumidores de dados que colaboram para promover a iniciativa. O ecossistema de Dados Abertos facilita a geração de novos negócios por meio da criação de sistemas e aplicativos inovadores voltados para a melhoria da produtividade das empresas existentes ou instituições governamentais, bem como para a melhoria do bem estar dos cidadãos. A efetiva criação e consolidação de um ecossistema de Dados Abertos depende, fundamentalmente, de uma boa comunicação entre os diferentes atores participantes do ecossistema. Dessa forma, no centro do ecossistema encontram-se uma plataforma de TI e um conjunto de melhores práticas, os quais têm como função facilitar a interação entre os diferentes atores, além de viabilizar a criação em larga escala de produtos e serviços inovadores, por meio da utilização de normas, padrões e ferramentas comuns. Este projeto tem como objetivo geral contribuir para o desenvolvimento do ecossistema de Dados Abertos da cidade de Recife por meio da especificação de melhores práticas e implementação de uma plataforma de Dados Abertos como serviço que facilitem a interação entre os diferentes atores participantes do ecossistema. É importante destacar que os resultados obtidos com este projeto podem ser aplicados em outras instâncias governamentais, tanto municipais quanto estaduais. Tanto as melhores práticas quanto a plataforma de Dados Abertos como serviço podem ser usadas para o desenvolvimento e consolidação do ecossistema de Dados Abertos de outras cidades do estado de Pernambuco, contribuindo, dessa forma, para o desenvolvimento do ecossistema de Dados Abertos do estado.

Avaliação da aplicação de Convex Approach e Dynamic Time Wrapping para o reconhecimento de caracteres
Descrição: A área de reconhecimento ótico de caracteres (OCR, do inglês Optical Character Recognition) vem sendo estudada já a algumas décadas, tendo encontrado avanço significativos recentemente devido ao avanço da tecnologia e do poder de processamento dos computadores. Esses avanços já permitem ao usuário de OCR?s transformar textos físicos inteiros em arquivos editáveis, buscando diversas finalidades, desde o arquivamento de textos antigos até a criação de bibliotecas virtuais de imagens, utilizando algumas das palavras chaves de imagens contendo textos para a indexação. Todo esse avanço ainda tem um custo computacional razoavelmente grande, o que atrapalha a possibilidade de uso em larga escala para a escrita manual, pois a forma de cada letra varia de escritor para escritor. Sendo assim, na literatura a melhor forma para contornar esse problema é a discretização das imagens que formam as letras de forma a extrair o mínimo possível de características para classificação. Dessa forma, esse trabalho tem como proposta trazer para o domínio da escrita manual uma técnica recente de classificação de gestos dinâmicos que tem tido resultados expressivos. A Convex Approch apresenta como proposta a extração de um vetor mínimo de características de forma a facilitar a classificação, evitando problemas como transladação e escala de imagens. Para comprovar a eficácia desta técnica, o algoritmo de classificação Dynamic Time Warping foi escolhido pela sua capacidade de prototipação e inovação, pois mesmo sendo uma técnica recente já encontra resultados satisfatórios na literatura..
